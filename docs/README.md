# Documentation

This folder contains focused documentation for the unified `esl-psc` interface, including the beginner GUI walkthrough.

- `gui-quickstart.md`: beginner-friendly GUI install and first-run guide
- `commands/pairs.md`: automatic pair selection with `esl-psc pairs`
- `commands/site-counter.md`: site counting with `esl-psc site-counter`

Project-wide installation notes, demos, and overview material remain in the repository [README.md](../README.md).
