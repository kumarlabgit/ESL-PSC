# ESL-PSC Demo Data

This archive contains small example inputs for learning ESL-PSC.

## Photosynthesis demo

Use this demo for the GUI quickstart and instructional videos.

- Alignments folder: `photosynthesis/alignments`
- Tree file: `photosynthesis/photo_tree.nwk`
- Phenotype file: `photosynthesis/photo_species_phenotypes.txt`
- Species groups file: `photosynthesis/photo_single_LC_matrix_species_groups.txt`

The photosynthesis example is small enough to run quickly on a laptop.

## Echolocation demo

The echolocation folder contains compact setup files and a compressed alignment
archive:

- Alignments archive: `echolocation/orthomam_echo_species_alignments.zip`
- Species groups file: `echolocation/orthomam_echo_species_groups.txt`

Unzip `orthomam_echo_species_alignments.zip` before selecting the echolocation
alignment folder in the GUI.
